# FIO-FromAToB-Modbus

## Factory I/O Information

- Scene: `From A to B`
- Brief description: A basic transport scene where the goal is to move a box from point A to point B until it reaches a sensor.
- Control protocol: Modbus (Factory I/O communication is handled via Modbus).
- Official scene details: https://docs.factoryio.com/manual/scenes/from-a-to-b/
